原型总览（iframe 平铺）

热力图绘画 · WeChat Mini Program Prototype

说明：本页面不使用跳转链接，直接用 iframe 平铺展示所有界面。 子页面使用 Tailwind CSS + TDesign 风格（圆角、色彩、层级、按钮）。

页面清单

新建、继续编辑、导出、更多操作占位
home.html
绘画编辑（editor）
画布、画笔参数、撤回/重做/清空/保存
editor.html
设置（settings）
授权说明、清理缓存、关于（占位）
settings.html
备注

  • 下一步如果要更“真”，可以把按钮点击做成纯前端的弹窗/ActionSheet 交互（仍不跳转）。
